# CarPooling Frontend

Modern React + TypeScript frontend for the CarPooling application.

## Tech Stack

- **Framework**: React 19 with Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: React Context API
- **Routing**: React Router v7
- **HTTP Client**: Axios
- **Build Tool**: Vite

## Getting Started

### Installation

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Copy environment variables:
   ```bash
   cp .env.example .env
   ```

### Development

Start the development server:
```bash
npm run dev
```

The app will be available at http://localhost:3000

See the full README in the repository for more details.
